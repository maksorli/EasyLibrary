# EasyLibrary

Программа "Управление библиотекой" — это консольное приложение для управления коллекцией книг. Приложение позволяет:

1. Добавлять книги.
2. Удалять книги.
3. Искать книги по заданным критериям.
4. Просматривать список всех книг.
5. Изменять статус книги (например, "в наличии" или "выдана").
6. Завершать работу программы.

Программа поддерживает интерактивный ввод с валидацией данных и предоставляет удобный интерфейс для взаимодействия с пользователем. Данные хранятся в *.json файле

---

## **Функциональные возможности**

- **Добавление книги**:
  - Ввод названия, автора и года издания.
  - Проверка корректности введённых данных (строки, числовые значения, диапазон годов).

- **Удаление книги**:
  - Удаление книги по её ID.

- **Поиск книги**:
  - Поиск книги по одному из критериев:
    - Название
    - Автор
    - Год издания
  - Отображение результатов поиска в виде таблицы.

- **Просмотр всех книг**:
  - Отображение всех книг в библиотеке в виде таблицы.

- **Изменение статуса книги**:
  - Возможность изменить статус книги:
    - "в наличии"
    - "выдана"

- **Выход из программы**:
  - Завершение работы программы.

# Описание директорий и файлов

### controllers/: Содержит контроллеры приложения.

- library.py: Основной класс Library для управления библиотекой.

- validators.py: Функции валидации пользовательского ввода.

### exceptions/: Пользовательские исключения.

-  exceptions.py: Определение собственного исключения ExitException.
### models/: Модели данных.

- book.py: Класс Book, представляющий книгу.

### storage/: Модуль для работы с хранилищем данных.

- json_storage.py: Класс JSONStorage для сохранения и загрузки данных в/из JSON-файла.
- library.json: Файл хранения данных библиотеки.
### tests/: Тесты для приложения.

- test_library.py: Тесты для класса Library.
- test_validators.py: Тесты для функций валидации.

### utils/: Вспомогательные функции и утилиты.

- menu.py: Функция для отображения меню.



#### config.py: Конфигурационные данные, такие как меню и критерии валидации

#### main.py: Точка входа в приложение.

#### Dockerfile: Конфигурационный файл для создания Docker-образа.

#### pre-commit-config.yaml: Конфигурация для pre-commit хуков.

# Использование
#### После запуска приложения вы увидите главное меню:



```
1. Добавить книгу
2. Удалить книгу
3. Найти книгу
4. Показать все книги
5. Изменить статус книги
6. Выход
Выберите действие:
```

Выберите нужное действие, введя соответствующий номер.

#### Добавление книги
- Выберите пункт 1 в главном меню. И последовательн введите данные в пункты меню:
```
Введите название книги.
Введите автора книги.
Введите год издания книги.
```
Примечания:

Название и автор должны содержать от 2 до 100 символов.
Год издания должен быть числом от 0 до текущего года.
Данные криерии можно изменить в файле  config.py

#### Удаление книги
- Выберите пункт 2 в главном меню.
- Введите ID книги, которую хотите удалить.

Если книга с указанным ID существует, она будет удалена из библиотеки, в противном случае появится сообщение об ошибке.

#### Поиск книги
- Выберите пункт 3 в главном меню.
```
Выберите критерий поиска:
1. По названию
2. По автору
3 .По году издания
4. Назад
```
Введите запрос для поиска.
Результаты поиска будут отображены в таблице.

#### Просмотр всех книг
- Выберите пункт 4 в главном меню, чтобы увидеть список всех книг в библиотеке.

#### Изменение статуса книги
- Выберите пункт 5 в главном меню.
```
Введите ID книги.
Выберите действие:
1. В наличии
2. Выдана
3. Назад

```
Статус книги будет обновлен.
